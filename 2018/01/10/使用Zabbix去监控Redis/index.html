дәҶи§ЈRedisзҡ„info

иҰҒиҺ·еҫ—Redisзҡ„еҪ“еүҚжғ…еҶөпјҢдҪҝз”Ёinfoе‘Ҫд»ӨеҚіеҸҜгҖӮе…·дҪ“з”Ёжі•пјҡredis-cli -h 127.0.0.1 -p 6379 -a redis_passwd info [еҸӮж•°] гҖӮй’ҲеҜ№дёҚеҗҢзҡ„еҸӮж•°е°ұдјҡзңӢеҲ°е…·дҪ“зҡ„ж•°еӯ—пјҢеҰӮжһңжІЎжңүеёҰеҸӮж•°пјҢйӮЈд№Ҳе°ұдјҡжҠҠй»ҳи®Өжғ…еҶөеҶҷеҮәжқҘпјҢеҰӮжһңеёҰдёҠallеҸӮж•°пјҢйӮЈд№Ҳе°ұдјҡжҠҠжүҖжңүжғ…еҶөйғҪеҶҷеҮәжқҘгҖӮжҜ”еҰӮпјҡredis-cli -h 127.0.0.1 -p 6379 -a redis_passwd info serverпјҢе°ұдјҡзңӢеҲ°redisе…ідәҺserverзҡ„дёҖдәӣж•°жҚ®пјҢеҰӮдёӢпјҡ

еҸҜд»ҘзңӢеҮәпјҢд»ҺserverйҮҢеҸҜд»ҘжҹҘиҜўеҲ°зҡ„жҳҜзүҲжң¬еҸ·гҖҒpidеҸ·гҖҒй…ҚзҪ®ж–Үд»¶и·Ҝеҫ„зӯүзӯүдёңиҘҝгҖӮ

еҰӮжһңеҸӮж•°жҳҜclientпјҢи®°еҪ•дәҶжҳҜе®ўжҲ·з«Ҝзҡ„зӣёе…ідҝЎжҒҜпјҡ

еҰӮжһңеҸӮж•°жҳҜmemoryпјҢи®°еҪ•зҡ„жҳҜеҶ…еӯҳзҡ„зӣёе…ідҝЎжҒҜпјҡ

иҝҷйҮҢиҰҒжіЁж„ҸпјҒеңЁзҗҶжғіжғ…еҶөдёӢпјҢ used_memory_rss зҡ„еҖјеә”иҜҘеҸӘжҜ” used_memory зЁҚеҫ®й«ҳдёҖзӮ№е„ҝпјҲжҲ‘иҝҷдёӘжңәеҷЁе°ұе·Із»ҸеұһдәҺдёҘйҮҚзҡ„зә§еҲ«дәҶпјүгҖӮеҪ“ rss > used пјҢдё”дёӨиҖ…зҡ„еҖјзӣёе·®иҫғеӨ§ж—¶пјҢиЎЁзӨәеӯҳеңЁпјҲеҶ…йғЁжҲ–еӨ–йғЁзҡ„пјүеҶ…еӯҳзўҺзүҮгҖӮеҶ…еӯҳзўҺзүҮзҡ„жҜ”зҺҮеҸҜд»ҘйҖҡиҝҮ mem_fragmentation_ratio зҡ„еҖјзңӢеҮәгҖӮеҪ“ used > rss ж—¶пјҢиЎЁзӨә Redis зҡ„йғЁеҲҶеҶ…еӯҳиў«ж“ҚдҪңзі»з»ҹжҚўеҮәеҲ°дәӨжҚўз©әй—ҙдәҶпјҢеңЁиҝҷз§Қжғ…еҶөдёӢпјҢж“ҚдҪңеҸҜиғҪдјҡдә§з”ҹжҳҺжҳҫзҡ„е»¶иҝҹгҖӮ

еҰӮжһңеҸӮж•°жҳҜstatsпјҢйӮЈе°ұжҳҜз»ҹи®Ўзҡ„зӣёе…ідҝЎжҒҜпјҡ

еҰӮжһңеҸӮж•°жҳҜCPUпјҢйӮЈд№Ҳе°ұдјҡиҝ”еӣһCPUзҡ„зӣёе…ідҝЎжҒҜпјҡ

еҰӮжһңеҸӮж•°жҳҜkeyspaceпјҢйӮЈд№Ҳе°ұдјҡиҝ”еӣһж•°жҚ®еә“зӣёе…ізҡ„з»ҹи®ЎдҝЎжҒҜпјҡ

йҷӨдәҶд»ҘдёҠд№ӢеӨ–е…¶д»–иҝҳжңүжӣҙеӨҡдҝЎжҒҜпјҢиҜ·з§»жӯҘпјҡhttp://redisdoc.com/server/info.html гҖӮж„ҹи°ўеүҚдәәж Ҫж ‘пјҒпјҒпјҒ

дҪҝз”Ёzabbixзӣ‘жҺ§redis

з”Ёzabbixзӣ‘жҺ§redisжҳҜдёҖдёӘеҫҲз®ҖеҚ•зҡ„дәӢе„ҝпјҢеҸӘйңҖиҰҒжҠҠйңҖиҰҒзӣ‘жҺ§зҡ„ж•°жҚ®жҸҗеҸ–еҮәжқҘеҚіеҸҜгҖӮиҖҢжҸҗеҸ–ж•°жҚ®зҡ„ж–№жі•е°ұжҳҜеҲ©з”ЁinfoеҺ»еҫ—еҲ°еҜ№еә”зҡ„ж•°еҖјгҖӮ

йҰ–е…Ҳе…ҲжқҘдёҖдёӘеҲӨж–ӯredisжңҚеҠЎеҷЁиҝһжҺҘзҡ„и„ҡжң¬пјҡ

з„¶еҗҺжӣҙж”№zabbix_agentd.conf,еҰӮдёӢпјҡ

дҝ®ж”№/etc/sudoersж–Үд»¶еҰӮдёӢпјҡ

дҝқеӯҳд№ӢеҗҺпјҢйҮҚеҗҜzabbix-agentжңҚеҠЎпјҢз”ұдәҺжҲ‘иҝҷдёӘredisжҳҜйҖҡиҝҮzabbix-proxyзӣ‘жҺ§зҡ„пјҢжүҖд»ҘеңЁzabbix-proxyдёҖз«Ҝз”Ёzabbix_getжқҘжҹҘзңӢз»“жһңпјҡ

з„¶еҗҺеңЁzabbix-proxyзҡ„жЁЎжқҝйҮҢйқўж·»еҠ дёҖдәӣйңҖиҰҒзӣ‘жҺ§зҡ„itemеҚіеҸҜпјҢжңүеҝ…иҰҒзҡ„иҜқеҸҜд»Ҙи®ҫзҪ®trigger+actionз”ЁжқҘжҠҘиӯҰпјҢеҰӮеӣҫпјҡ
