不过这个脚本有两个缺点：
第一：如果阿里云服务器是中文名称，那么使用api查询出现的是十六进制的符号；
第二：如果服务器里没有外网IP或者是后开的临时带宽，那么在outIP的地方得到的值是空，sql语句会因为少一项而报错；
第三：这个api没有查询服务器带宽值的功能，还需要另外写一个脚本搭配。
